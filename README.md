# challenge10_coderhouse -- 

## Despliegue 📦

* Instalacion de depedencias
```bash
npm i 
```
* Modificar el string de conexion de mongo local para poderse conectar. 

* Ejecucion del proyecto handlebars (se ejecuta en puerto 8080) :godmode:
```bash
npm run start
```
